Реализованные breakpoints (Desktop First): В проекте реализовано 2 основных контрольных точки (breakpoints) для адаптации интерфейса:

1024px: Адаптация для планшетов и небольших ноутбуков. Изменяется Grid-сетка главного экрана (уменьшение отступов, перестройка колонок hero-блока), чтобы контент помещался без горизонтального скролла.

768px: Адаптация для мобильных устройств.

Главный экран перестраивается в одну колонку (display: block).

Навигационное меню перемещается в общий поток документа.

Товарные сетки (Grid) перестраиваются в 1 колонку для удобного чтения.

Отключаются сложные декоративные наложения текста.

Обоснование выбора методологии: Для организации CSS-кода была выбрана методология БЭМ/

Причина выбора: БЭМ позволяет создавать независимые блоки, которые легко переиспользовать и стилизовать, не опасаясь конфликтов специфичности. Это упрощает поддержку кода и делает структуру классов понятной/
